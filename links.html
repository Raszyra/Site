External Links

Here's a collection of all the best software from around the web. If there's anything missing from this list, send me the link - I love trying out new tools.



GitHub/Github Pages

Where we are now! GitHub is a mildly-complicated filehosting site with a truly insane amount of functionality - made for sharing codebases amongst multiple employees.
GitHub Pages allows free sitehosting from any of your file repositories. You can see this site's files if you visit mine at github.com/Raszyra/Site .
- Visit the GitHub website -

Code::Blocks

A full-featured, free and open-source C++ IDE. Comes with MinGW GCC for compiling C++ code.
This is where I started programming - it's pretty easy to use and well supported.
- Visit the Code::Blocks website -

CxxDroid

A handy C++ IDE & compiler for Android. You'll need a paid subscription to export to APK, but you can write and compile C++ on the go for free (this is how I made The Silver Flask originally.)
- CxxDroid on the Google Play Store -

Pivot Animator

A joint-rotation-based animation software with plenty of base functionality, and a massive user-made asset library.
Pivot makes positioning and animating simple characters the work of minutes (perfect for if you need to animate a short sequence).
- Visit the Pivot website -

OBS Studio

The easiest and free-est screen capture software I could find. Absolutely simple to use and does exactly what it says on the box.
- Visit the OBS Project website -

ChatGPT

No way you don't know what this is...and if you don't, I'm not going to spoil it.
(GPT4 wrote most of the code for this website. Man, I love the future.)
- Visit the ChatGPT website -

Printful

A crazy-easy, market driven market maker - they'll host your online storefront, work with the suppliers and distributors, and provide basic marketing tools for free, leaving you a cut that you choose of inventory-less profit.
- Visit the Printful website -
